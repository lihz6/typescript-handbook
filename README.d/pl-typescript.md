## Basic Types

- `boolean`: `true` `false`
- `number`: `0` `0.0`
- `string`: `''` `""` <code>\`\${expr}`</code>
- `array`: `Array<T>` `T[]`
- `tuple`: `[T1, T2, ...]`
- `enum`: `enum { ... }` `const enum { ... }`
- `any`
- `void`
- `null`
- `undefined`
- `never`
- `object`

## Type Assertions

- `<Type>value`
- `value as Type`
- `value!`

## Variable Declarations

- `var`
- `let`
- `const`
